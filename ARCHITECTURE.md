## Data Flow

### Public User (View Projects)
1. User visits http://localhost:3000/projects
2. React app fetches data from /api/projects/
3. Django returns JSON with project list
4. React Query caches the data
5. ProjectCard components render projects

### Admin User (Add Project)
1. Admin logs in at /admin/login
2. Frontend sends credentials to /api/token/
3. Backend validates and returns JWT tokens
4. Frontend stores tokens in localStorage
5. Admin accesses Django admin at /admin/
6. Creates project with images
7. Project saved to database
8. Frontend fetches updated project list

### Collaboration Request
1. User fills form at /collaborate
2. Form data sent to /api/collaborations/
3. Backend validates and saves to database
4. Success message shown to user
5. Admin can view in Django admin
